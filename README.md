# web-codigo-nativo-php

This project is a custom rebuild of the official Código Nativo website, developed as an optimized version using a PHP MVC architecture.

The goal is to replicate the current WordPress-based website while improving performance, structure, scalability and maintainability through a custom codebase.

---

## 🎯 Project Purpose

The main purpose of this project is to:

- Replace WordPress with a custom PHP MVC solution
- Improve performance and loading times
- Create a cleaner and more scalable architecture
- Gain full control over the frontend and backend logic

This project serves both as a production-ready alternative and a technical learning process.

---

## ✨ Features

- Custom MVC architecture in PHP
- Modular and reusable components
- SEO-friendly structure
- Performance-focused implementation
- Custom routing system

---

## 🛠️ Tech Stack

- PHP
- MVC Architecture
- HTML
- CSS
- JavaScript
- MySQL
